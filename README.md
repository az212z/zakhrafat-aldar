# زخرفة الدار — موقع تعريفي

موقع تسويقي ثابت (vanilla static) لمحل **زخرفة الدار** للديكور وورق الجدران والتشطيبات في الخبر.

## المعلومات
- **النشاط:** ديكور وورق جدران وتشطيبات
- **التصنيف:** HomeGoodsStore / GeneralContractor
- **المدينة:** الخبر — الخبر الشمالية، طريق الملك فهد
- **التقييم:** 4.8 على خرائط قوقل (114 مراجعة) — مُقتبس في الموقع وفي بيانات JSON-LD
- **التواصل:** جوال 0500943560 · واتساب 966500943560

## لماذا اختير هذا النشاط
نشاط غنيّ بالصور الحقيقية للتنفيذ (جدران، أسقف، تكسيات)، وبسمعة قوية (4.8 من 114 مراجعة)، ولا يملك موقعًا — وهو نشاط ذو نية شرائية واضحة (تشطيبات وديكور).

## فكرة التصميم — "Warm Craftsman"
هوية دافئة تركّز على الحِرفة والتفاصيل، متمايزة عن باقي مواقع الباتش:
- **الألوان:** طِيني/تراكوتا (`#B45B36`) + فحمي (`#2A2421`) + كريمي (`#F7F1E8`)، مع لمسة ذهبية للنجوم.
- **الخطوط:** عرض **El Messiri** + كوفي **Reem Kufi** للعناوين، و**Tajawal** للنصوص.
- **الطابع:** معرض أعمال يقود الصفحة (gallery-led)، مساحات واسعة، كروت بظلال خفيفة، حركة ناعمة محترمة لـ `prefers-reduced-motion`.

## حالة الـ backend
لا يوجد خادم. نموذج "طلب عرض سعر" **تجريبي**:
- يتحقق من المدخلات (الاسم، الجوال، الخدمة).
- يخزّن نسخة في `localStorage` (تجريبي فقط).
- يبني رسالة واتساب جاهزة ويفتح `wa.me/966500943560`.
- يعرض إشعار (toast) بالنجاح. لا يُرسَل شيء لأي خادم.
